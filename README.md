# ML Classification Project

## Overview

This project builds and evaluates machine learning models to classify breast tumors as benign or malignant.

## Technologies Used

* Python 3.x
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab / Jupyter Notebook

## Dataset

Breast Cancer Dataset from scikit-learn.

Target Classes:

* 0 = Malignant
* 1 = Benign

## Environment Setup

Install dependencies:

```bash
pip install pandas numpy matplotlib scikit-learn
```

## Running the Project

1. Open the notebook:

```bash
jupyter notebook
```

2. Run:

```text
ML_Classification_Project.ipynb
```

3. Execute all cells.

## Models Used

1. Logistic Regression
2. Random Forest Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC
* Cross Validation
